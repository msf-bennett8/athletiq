Installation Guide

  1. Download the APK file by clicking the download button above
  2. Go to your device Settings > Security (or Privacy)
  3. Enable "Install from Unknown Sources" or "Allow from this source"
  4. Open your file manager and locate the downloaded APK file
  5. Tap the APK file and follow the installation prompts
  6. Once installed, you can disable "Unknown Sources" for security
  7. Open the CoachPro app and start your coaching journey!
